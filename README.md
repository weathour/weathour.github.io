# texture

从thelehhman那里弄来的一个结构非常简单易懂的博客模型，又简洁又好看（最重要的是这个代码也非常简洁啊，看满屏英语头疼的人看着非常舒服）

**Demo**: [thelehhman.com/texture](https://thelehhman.com/texture)

这个是他的页面，想要模板的直接在github上查texture就行了。


**_config.yml**

- showPicker：[false|true] #这个是测试界面使用的
- disqus_shortname: games #这个是做评论区的
- analytics_id: '< YOUR ID >'#这个是写google analuticsID的
- show_excerpts: true #展示节选的
- showNav: true #home键
